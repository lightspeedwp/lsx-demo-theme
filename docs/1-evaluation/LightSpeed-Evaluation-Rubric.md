-   This rubric references the complete intern journey: block themes, plugin development, Figma/MCP, testing, production contributions, and communication/workflow.

-   Use logs, planning, reflections, project boards, PR reviews, and Figma files as evidence during evaluation.
-   Mentors should cross-reference intern work (logs, PRs, project boards, Figma files) with deliverables and checklist items monthly.

This scoring guide is authoritative for all intern evaluations at LightSpeed and ensures alignment between the curriculum, client-ready outcomes, and mentoring standards.
