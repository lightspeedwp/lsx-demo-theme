## 1. Global Text & Navigation Shortcuts

These shortcuts work across almost all text-editing environments, including Google Docs, GitHub Markdown files, and code editors.

-   Ctrl + → (Windows/Linux) / Option + → (Mac) — Move cursor one word to the right
-   Ctrl + ← (Windows/Linux) / Option + ← (Mac) — Move cursor one word to the left
-   Home / Ctrl + ← (Windows/Linux) / Cmd + ← (Mac) — Move cursor to the start of the current line
-   End / Ctrl + → (Windows/Linux) / Cmd + → (Mac) — Move cursor to the end of the current line
-   Shift + Home (Windows/Linux) / Shift + Cmd + ← (Mac) — Select text from cursor to the start of the line
-   Shift + End (Windows/Linux) / Shift + Cmd + → (Mac) — Select text from cursor to the end of the line
-   Ctrl + A (Windows/Linux) / Cmd + A (Mac) — Select all content
-   Ctrl + Z (Windows/Linux) / Cmd + Z (Mac) — Undo the last action
-   Ctrl + Y (Windows/Linux) / Cmd + Shift + Z (Mac) — Redo the last action

## 2. VS Code (Code Editor) Shortcuts

VS Code is our primary code editor. These shortcuts will dramatically improve your coding, refactoring, and file navigation speed.

-   Ctrl + P (Windows/Linux) / Cmd + P (Mac) — Go to File (Quick Open)
-   Ctrl + Shift + F (Windows/Linux) / Cmd + Shift + F (Mac) — Search across all files in the project
-   Ctrl + D (Windows/Linux) / Cmd + D (Mac) — Select next occurrence of the current selection (Multi-cursor editing)
-   Alt + ↑ (Windows/Linux) / Option + ↑ (Mac) — Move line up
-   Alt + ↓ (Windows/Linux) / Option + ↓ (Mac) — Move line down
-   Ctrl + / (Windows/Linux) / Cmd + / (Mac) — Toggle line comment
-   Ctrl + Shift + L (Windows/Linux) / Cmd + Shift + L (Mac) — Select all occurrences of the current selection
-   F1 / Ctrl + Shift + P (Windows/Linux) / Cmd + Shift + P (Mac) — Show Command Palette (access all VS Code commands)

## 3. Figma (Design & Handoff) Shortcuts

You will use Figma for design handoff and inspecting the LSX Design System. These shortcuts are essential for efficient measurement and asset extraction.

-   Shift + D — Toggle Dev Mode (Essential for handoff!)
-   Ctrl + / (Windows) / Cmd + / (Mac) — Open Quick Actions Search (Find any command)
-   Ctrl + Shift + R (Windows) / Cmd + Shift + R (Mac) — Show/Hide Rulers
-   Shift + 2 — Zoom to Fit
-   Ctrl + Shift + 4 (Windows) / Cmd + Shift + 4 (Mac) — Zoom to Selection

## 4. GitHub & Browser Navigation

Since we rely heavily on GitHub for workflow (Issues, PRs, Commits), these shortcuts help navigate quickly.

-   t (when viewing a repo) — GitHub: Open the file finder for quick file navigation
-   s or / — GitHub: Activate the search bar to find issues, repos, etc.
-   g then i — GitHub: Navigate to the Issues page of the repository
-   g then p — GitHub: Navigate to the Pull Requests page of the repository
-   Ctrl + Tab (Windows/Linux) / Cmd + Option + → (Mac) — Switch to the next browser tab
-   Ctrl + Shift + Tab (Windows/Linux) / Cmd + Option + ← (Mac) — Switch to the previous browser tab
