## 1. Heading style shortcuts

Use heading styles to keep documents accessible and machine-readable. These shortcuts apply in both Google Docs and many rich text editors.

| Style       |    PC Shortcut | Mac Shortcut | Purpose                                            |
| ----------- | -------------: | -----------: | -------------------------------------------------- |
| Normal Text | Ctrl + Alt + 0 |    ⌘ + ⌥ + 0 | Standard paragraph text. Use this as your default. |
| Heading 1   | Ctrl + Alt + 1 |    ⌘ + ⌥ + 1 | Main section titles (as used for this document).   |
| Heading 2   | Ctrl + Alt + 2 |    ⌘ + ⌥ + 2 | Primary subsection titles.                         |
| Heading 3   | Ctrl + Alt + 3 |    ⌘ + ⌥ + 3 | Secondary subsection titles.                       |

## 2. Text Formatting Basics

These shortcuts will save time when styling words and phrases for emphasis. Use formatting sparingly—too much formatting makes text harder to read.

| Action                   |      PC Shortcut |  Mac Shortcut | Note                                                                                  |
| ------------------------ | ---------------: | ------------: | ------------------------------------------------------------------------------------- |
| Bold                     |         Ctrl + B |         ⌘ + B | Used for emphasis, key terms, or section titles (if not using headings).              |
| Italics                  |         Ctrl + I |         ⌘ + I | Used for book titles, technical names, or when introducing a new concept.             |
| Underline                |         Ctrl + U |         ⌘ + U | Primarily used for linking; avoid using for emphasis.                                 |
| Copy                     |         Ctrl + C |         ⌘ + C |                                                                                       |
| Paste                    |         Ctrl + V |         ⌘ + V |                                                                                       |
| Paste without Formatting | Ctrl + Shift + V | ⌘ + Shift + V | Crucial for maintaining consistent document style when pasting from external sources. |
| Clear Formatting         |        Ctrl + \  |        ⌘ + \  | Removes bold, italics, font size, etc., reverting to Normal Text style.               |

## 3. Lists and Indentation

Clear, concise lists are vital for documentation. Prefer bulleted lists for non-sequential items and numbered lists for steps or ranked items.

Lists

-   Bulleted List: Ctrl + Shift + 8 (PC) / ⌘ + Shift + 8 (Mac)
-   Numbered List: Ctrl + Shift + 7 (PC) / ⌘ + Shift + 7 (Mac)

Indentation and Alignment

| Action          |      PC Shortcut |  Mac Shortcut | Purpose                                                  |
| --------------- | ---------------: | ------------: | -------------------------------------------------------- |
| Increase Indent |         Ctrl + ] |         ⌘ + ] | Creates nested list items or increases paragraph margin. |
| Decrease Indent |         Ctrl + [ |         ⌘ + [ | Decreases nested list items or reduces paragraph margin. |
| Center Align    | Ctrl + Shift + E | ⌘ + Shift + E | Rarely used; reserved for titles or brief quotes.        |
| Left Align      | Ctrl + Shift + L | ⌘ + Shift + L | Standard alignment for all body text.                    |

## Google Sheets Formatting and Data Presentation

Google Sheets is used for tracking time (Harvest), managing project budgets, analyzing sales data, and organizing curriculum. Proper formatting ensures data is readable and charts are accurate.

### 4. Essential Cell Formatting

Never manually type currency symbols or percentage signs—use the format button or shortcut.

| Format             |         PC Shortcut |   Mac Shortcut | Purpose                                                     |
| ------------------ | ------------------: | -------------: | ----------------------------------------------------------- |
| Currency (e.g., R) |    Ctrl + Shift + 4 |  ⌘ + Shift + 4 | Format numbers as currency. Essential for budgeting sheets. |
| Percentage (%)     |    Ctrl + Shift + 5 |  ⌘ + Shift + 5 | Format numbers as percentages.                              |
| Date               |    Ctrl + Shift + 3 |  ⌘ + Shift + 3 | Displays numerical data as a date.                          |
| Decimal Places     | Ctrl + Alt + . or , | ⌘ + ⌥ + . or , | Increase / Decrease decimal places for precision.           |

## 6. Creating and Editing Tables

Docs tables are static; Sheets tables are dynamic and should be used for data that requires calculation, sorting, or filtering.

Key Table Features

-   Freeze Rows/Columns: Keep header rows visible while scrolling — View > Freeze. Freeze at least the first row (shortcut: Ctrl + Alt + F, R on PC).
-   Filter / Sort: Temporarily reorganise data without changing the source order: Select the range → Funnel Icon, or use Ctrl + Shift + L (PC) / ⌘ + Shift + F (Mac).
-   Conditional Formatting: Automatically change cell background/text color based on cell value (e.g., green if under $500). Access via Format > Conditional formatting.
